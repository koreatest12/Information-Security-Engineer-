# Data Shield Workflow Factory

이 디렉토리는 민감 데이터의 암호화·복호화, 외부 유출 방지(DLP), 데이터베이스 테이블 암호화 및 인덱스 생성을 자동화하기 위한 템플릿을 저장합니다. GitHub Actions 워크플로우를 통해 대량으로 생성되며, 데이터 보호 작업을 반복 가능하게 유지하는 것을 목표로 합니다.

## 포함되는 보안 단계
- **암호화/복호화**: AES-256-GCM 알고리즘을 기본으로 하여 콘텐츠를 보호하고 검증합니다.
- **외부 유출 방지**: 파일 확장자와 MIME 기반의 DLP 필터로 유출 가능성을 최소화합니다.
- **DB 암호화 및 인덱스**: 테이블 단위 컬럼 암호화와 해시 기반 인덱스를 추가하여 검색 성능과 보안을 모두 확보합니다.

## 사용 방법
1. `.github/workflows/data-shield-factory.yml` 워크플로우를 수동 실행하거나 스케줄에 따라 대량 템플릿을 생성합니다.
2. 생성된 템플릿(`data_shield_workflow_XX.yml`)을 참조하여 서비스별 파이프라인에 맞게 키·테이블 이름·규칙을 조정합니다.
3. 템플릿은 idempotent하게 갱신되므로 반복 실행 시에도 최신 보안 기준을 반영합니다.
